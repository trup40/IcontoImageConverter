# Icon to Image Converter 🎨

"ancient voices my lord"

A comprehensive web-based tool that allows users to select icons from various popular libraries, customize their color and size, and download them as **PNG** (transparent) or **JPEG** files instantly.

This project runs entirely in the browser using client-side technologies, eliminating the need for server-side image processing.

## 🌟 Features

* **Massive Icon Library:** Access to over 15+ icon sets in one place.
* **Real-time Customization:**
    * 🎨 **Color Picker:** Change icon, background, and canvas colors instantly.
    * 📏 **Size Slider:** Adjust icon size from small web icons to high-res images (up to 2500px).
* **Export Options:**
    * Download as **PNG** (Transparent background).
    * Download as **JPEG** (White or Black background).
* **Smart Canvas Rendering:** Includes specific algorithms to handle font offsets and rendering quirks for complex icon sets like Weather Icons.

## 🚀 How to Use

Since this is a static web application, no build process or backend server is required.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/icon-to-image-converter.git](https://github.com/YOUR_USERNAME/icon-to-image-converter.git)
    ```
2.  **Open the file:**
    Navigate to the project folder and open `index.html` in any modern web browser.
3.  **Start converting:**
    * Select an icon set from the dropdown menu.
    * Search for an icon.
    * Adjust the color and size.
    * Click the "Download" buttons.
